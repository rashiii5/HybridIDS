What is SHAP?

SHAP (SHapley Additive exPlanations) assigns each feature an importance value for a specific prediction.


■ Red bars — features that push toward the predicted class.


■ Cyan bars — features that push away from the predicted class.


Latent features (latent_0 … latent_7) are the VAE's compressed representation of the traffic pattern.
